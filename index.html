SecureGraph

SecureGraph api similar to Blueprints but secure

Download this project as a .zip file Download this project as a tar.gz file

Secure Graph is an API to manipulate graphs, similar to blueprints. Unlike blueprints, every Secure graph method requires authorizations and visibilities. Secure graph also supports multivalued properties as well as property metadata.

The Secure Graph API was designed to be generic, allowing for multiple implementations. The only implementation provided currently is built on top of Apache Accumulo for data storage and Elastic Search for indexing.
